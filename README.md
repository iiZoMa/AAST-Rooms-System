AAST Hall Booking System - Operational Overview
The AAST Hall Booking System is a premium, centralized platform designed to streamline the reservation and management of academic halls, laboratories, and multipurpose rooms across the academy. Here is how the system operates:

1. Secure Authentication & Role-Based Access
The system employs a secure login mechanism using Employee IDs. Users are assigned specific roles (Admin, Branch Manager, Dean, Secretary, or Employee), each with a tailored dashboard and specific permissions:

Admins: Have full control over the system, including master registries, global overrides, and employee databases.
Managers/Deans: Oversee departmental bookings and handle approval workflows.
Employees: Can search for spaces, submit booking requests, and manage their personal schedules.
2. Intelligent Room Discovery
Users can navigate to the "Find Space" section to view real-time availability of rooms. The system filters spaces by type (Academic Blocks vs. Special Facilities) and capacity, ensuring that users find the most suitable location for their needs.

3. Streamlined Booking Workflow
Request Submission: Users select a time slot and provide details such as the purpose of the booking and expected attendance.
Approval Cycle: Requests are automatically routed to the relevant authority (e.g., Branch Manager or Admin). Users receive real-time notifications regarding the status of their requests (Pending, Approved, or Rejected).
Multipurpose Requests: A specialized form is available for complex event needs, including technical requirements and staff support.
4. Schedule Management
The "My Bookings" portal provides a comprehensive overview of all upcoming and past reservations. It features a split-view design that allows users to monitor their "Booking Summary" (total hours, monthly count) while managing active requests.

5. Advanced Administrative Tools
For high-level management, the system includes:

Master Registry: A centralized console to monitor all campus-wide activities.
Global Overrides: Admins can manually adjust bookings or create custom time slots outside standard hours.
Daily Activity Reports: Automated system reports tracking all changes made within the last 24 hours.
6. Premium User Experience
Built with a modern Dark Navy & Gold aesthetic, the system offers a premium "Portal" feel. It supports Bilingual Functionality (Arabic & English) with a seamless RTL/LTR toggle, ensuring accessibility for all staff members.
